
# Portafolio de Joao Alexander Barres Díaz

¡Hola! 👋 Soy Joao, desarrollador web apasionado por crear experiencias digitales modernas y funcionales. Este es mi portafolio personal donde muestro mis proyectos, habilidades y tecnologías favoritas.

## 🚀 Descripción
Este portafolio está construido con tecnologías de última generación y muestra proyectos reales, animaciones, integración con APIs y un diseño responsive.

## 🛠️ Tecnologías principales

- **Next.js** (React)
- **TailwindCSS**
- **Framer Motion**
- **EmailJS**
- **Flowbite**
- **HeroUI**

## 🌐 Idioma
- Español

## 📦 Instalación y uso

1. Clona el repositorio:
	```bash
	git clone https://github.com/Joaokiller360/portfolio.git
	cd portfolio
	```
2. Instala las dependencias:
	```bash
	npm install
	```
3. Crea un archivo `.env.local` y configura tus variables de entorno necesarias (por ejemplo, `NEXT_PUBLIC_BACKND_URL`).
4. Inicia el servidor de desarrollo:
	```bash
	npm run dev
	```
5. Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## 🖥️ Despliegue
Puedes desplegar este portafolio fácilmente en plataformas como Vercel, Netlify o tu propio servidor.
